使用

tools.strs.[methods]

方法

方法名 说明 用法 返回值
byteSize(str) 返回字符串的字节长度 strs.byteSize('Hello World') || strs.byteSize('😀sdfs') Number
stripHTMLTags(str) 删除字符串中的HTMl标签 strs.stripHTMLTags('<p><em>lorem</em> <strong>ipsum</strong></p>') => 'lorem ipsum' -
changeCase(str, type = 1) 字符转换

type: 1:每个单词首字母大写 2：每个单词首字母小写 3：大小写转换 4：全部大写 5：全部小写 6：首字母大写 7: 首字母小写

strs.changeCase('Hello World') String
trim(str, type = 1) 去除空格,

type: 1-所有空格 2-前后空格 3-前空格 4-后空格

strs.trim(' ada asdas ') String

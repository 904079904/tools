使用

tools.funs.[methods]

方法

方法名 说明 用法 返回值
defer(fn[, ...args]) 推迟执行 funs.defer(console.log, 'a'), console.log('b') => logs 'b' then 'a' -
toHump(str[, s = '-']) 转驼峰 funs.toHump('a-b') || funs.toHump('a&b', '&') -
debounce({callback = null, reset = null, seconds = 100}) 防抖, 一定时间内未触发在执行 funs.debounce({
callback: () => console.log('函数执行, 你的程序逻辑'),
reset: () => console.log('失败提示')
}) 		-
throttle({callback = null, reset = null, seconds = 1000}) 节流 一定时间段内只调用一次事件处理函数 funs.throttle({
callback: () => console.log('函数执行, 你的程序逻辑'),
reset: () => console.log('失败提示')
}) 		-
disconnection({callback = null, reset = null, seconds = 1000}) 一定时间内触发不执行， 等时间过去在激活 funs.throttle({
callback: () => console.log('函数执行, 你的程序逻辑'),
reset: () => console.log('失败提示')
}) 		-
